    BlockCheck

    проверка блокировки без обхода

    Проверяет доступность доменов/IP из списка напрямую: тестовое соединение исключается из основного сервиса nfqws2 и не обходится, поэтому видно, что реально блокирует провайдер (RST, таймаут, обрыв на ~16 КБ).
