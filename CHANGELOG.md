# Changelog

Все заметные изменения проекта. Формат — [Keep a Changelog](https://keepachangelog.com/ru/),
версионирование приближено к SemVer.

## [v0.1.3] — 2026-05-28
### Добавлено
- **Авторизация веб-интерфейса** по логину/паролю — проверка против системного
  аккаунта роутера (`/opt/etc/passwd` / `/etc/shadow`, crypt), как у
  nfqws-keenetic-web. Сессия в cookie, выход, переключатель в настройках
  (по умолчанию включена). Аварийное отключение через `N2S_NOAUTH=1`.
- **Выбор блобов для прогона** — в настройках прогона можно отметить блобы
  (системные и пользовательские); выбранные загружаются во все тестируемые
  стратегии (`--blob=имя:@путь`), чтобы стратегии могли на них ссылаться.
- **GeoSite/GeoIP** — загрузка файла `geosite.dat`/`geoip.dat` (формат v2ray)
  или текстового списка, просмотр категорий и импорт категории (с лимитом) в
  тестовый список.

## [v0.1.2] — 2026-05-28
### Добавлено
- Тёмная тема: по умолчанию следует настройке браузера, переключатель в шапке
  (авто / светлая / тёмная) и override через `?theme=`.
- Удаление списка прямо из карточки в боковой панели.
- Загрузка блобов через drag & drop со стилизованным выбором файла.
### Исправлено
- Устаревший UI после самообновления: ссылки на ассеты версионируются
  (`?v=<версия>`) и отдаются с `Cache-Control: no-store`.

## [v0.1.1] — 2026-05-28
### Добавлено
- Самообновление из интерфейса (проверка GitHub Releases, скачивание, замена
  бинарника и перезапуск сервиса).
- Полный редизайн интерфейса в стиле Keenetic OS с анимациями.

## [v0.1.0] — 2026-05-28
### Добавлено
- Первый релиз: веб-инструмент автоподбора рабочих стратегий обхода DPI для
  nfqws2 (zapret2) — списки доменов/IP, многопоточные изолированные прогоны,
  метрики (задержка / скорость / коэффициент) с сортировкой, пользовательские
  блобы, применение стратегии в конфиг, установщик, мульти-арч сборки.
